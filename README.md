# Dimitris Angelopoulos — Portfolio

Personal portfolio website built with plain HTML, CSS, and JavaScript. Showcases my projects, skills, and background as a graduate student in Information Technology.

**Live site:** https://dimitris232989.github.io

## Projects featured

- **Pivai Tech** — Company website I designed and built as an IT Assistant (live in production)
- **Euroleague Stats App** — Full-stack PHP/MySQL web application built for a Database Systems course
- **RestoreX** — UX/UI design project built in Figma for a Human-Computer Interaction course
- **FitLog** — Fitness tracking app designed in Figma and developed in Microsoft Power Apps

## Tech stack

- HTML / CSS / JavaScript
- Hosted on GitHub Pages

## Run locally

Clone the repo and open `index.html` in your browser. No build step or dependencies required.

```bash
git clone https://github.com/dimitris232989/dimitris232989.github.io.git
cd dimitris232989.github.io
```
